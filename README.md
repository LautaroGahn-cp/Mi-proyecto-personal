# Mi pagina personal

## Descripción
Esta pagina se trata sobre una pequeña guia al momento de jugar el videojuego New Super Mario Bros de la consola Nintendo Wii.

## Contenido
- Menu principal en donde mostramos una pequeña informacion y enlazamos hacia las diferentes secciones de la pagina.
- Contiene tabla de Informativa.
- Contine formulario de contacto.
- Contiene un Gameplay.
- Contiene assets.
